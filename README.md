# TECHCUP FUTBOL - ECI

Digital platform for managing the semester football tournament of the Systems Engineering,
Artificial Intelligence Engineering, Cybersecurity Engineering, and Statistical Engineering
programs at the [Escuela Colombiana de Ingeniería Julio Garavito](https://www.escuelaing.edu.co/es/).

*Note*: If you want to know further information, please read the requirements file which
can be found in *docs/requirements.pdf*.
