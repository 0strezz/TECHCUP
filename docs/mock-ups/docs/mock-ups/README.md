# Mock Ups - TechCup Futbol

This document contains the main user interface mockups for the **TechCup Futbol platform**, a system designed to manage the internal football tournament at Escuela Colombiana de Ingeniería.

The following screens represent the main interactions available to users within the platform.

## Designed Screens

### Login Screen
Allows users to authenticate into the system using their credentials.

![Login](./images/login.png)

### User Registration
Enables players to create a new account and register their basic information in the platform.

![User Registration](./images/user-registration.png)

### Player Profile
Allows users to configure their personal and football-related information.

![Player Profile](./images/player-profile.png)

### Matches
Displays the list of current matches available for the team.

![Matches](./images/matches.png)

### Match History
Displays the list of previous matches and their results.

![Match History](./images/match-history.png)

### Statistics
Shows player statistics and performance information.

![Statistics](./images/statistics.png)

### Create Team
Allows a team captain to create a new team, define the team name, upload a logo, and configure uniform colors.

![Create Team](./images/create-team.png)

### Upload Payment Receipt
Allows captains to upload the payment receipt required to register the team in the tournament.

![Upload Payment Receipt](./images/upload-payment-receipt.png)

### Standings
Displays the tournament standings including matches played, wins, draws, losses, goals scored, goals conceded, and points.

![Standings](./images/standings.png)
